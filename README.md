# Projet Student API - Spring Boot
## Introduction
Ce projet a pour objectif de développer une **API REST complète** pour la gestion des étudiants, en utilisant **Spring Boot**.L'application permet 
d'effectuer toutes les opérations CRUD (Create, Read, Update, Delete) sur une entité `Student`, avec une architecture propre et bien structurée.

### Objectifs pédagogiques
- Créer un projet Spring Boot avec les dépendances nécessaires (Web, JPA, MySQL, Validation, Lombok, Swagger)
- Configurer la connexion à une base de données MySQL
- Modéliser une entité JPA
- Utiliser Spring Data JPA pour l'accès aux données
- Séparer les entités des DTO (Data Transfer Objects) en utilisant des **records Java**
- Implémenter un service métier
- Créer un contrôleur REST exposant les endpoints CRUD
- Gérer les erreurs de manière globale
- Tester et documenter l'API avec **Swagger UI**
- Valider le bon fonctionnement avec des tests manuels via Swagger et phpMyAdmin
